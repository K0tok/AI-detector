# AI Text Detector for DOCX Files (Russian)

This program analyzes DOCX documents to detect text that may have been generated by AI tools. It identifies common patterns in AI-generated text and provides suggestions for making the text appear more human-written. The program is designed to work with Russian language text.

## Features

- Detects AI-generated text patterns in DOCX files
- Identifies overly formal language, repetitive structures, and formulaic phrases in Russian
- Estimates the probability that text was AI-generated
- Provides specific suggestions for improvement in Russian
- Can generate a "fixed" version of the document with more natural language
- Saves analysis reports in JSON format

## Usage

### Command Line

```bash
# Basic analysis
python ai_detector.py document.docx

# Save analysis report
python ai_detector.py document.docx -o report.json

# Generate improved document
python ai_detector.py document.docx --fix --fix-output improved.docx
```

### As a Module

```python
from ai_detector import AITextDetector

detector = AITextDetector()
results = detector.process_docx_file("example_russian.docx")

print(f"AI Probability: {results['analysis']['ai_probability']}%")
print(f"Issues found: {len(results['analysis']['issues'])}")
```

## How It Works

The detector looks for common patterns in AI-generated text:

- Overly formal language (e.g., "осуществлять" instead of "делать")
- Repetitive transition phrases (e.g., "Более того", "Кроме того")
- Formulaic sentence structures (e.g., "Во-первых", "Во-вторых", "В-третьих")
- Common AI phrases (e.g., "В современном мире", "Следует отметить")

## Ethical Considerations

This tool is designed to help users improve their writing and make AI-assisted content appear more natural. It should be used responsibly and ethically, respecting academic and professional integrity standards.

## Russian Language Support

The program specifically targets Russian language patterns that are common in AI-generated text. It includes:

- Detection of formal Russian words and phrases
- Recognition of repetitive sentence structures typical in Russian AI text

- Text fixing capabilities to replace formulaic phrases with more human-like alternatives
