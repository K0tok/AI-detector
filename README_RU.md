# Детектор текста ИИ для DOCX файлов (Русский)

Программа анализирует DOCX документы для выявления текста, который мог быть сгенерирован инструментами ИИ. Она определяет общие паттерны в тексте, созданном ИИ, и предоставляет рекомендации по улучшению текста, чтобы он выглядел более естественным. Программа разработана для работы с русским языком.

## Возможности

- Обнаруживает паттерны текста, созданного ИИ, в DOCX файлах
- Определяет чрезмерно формальный язык, повторяющиеся структуры и шаблонные фразы на русском языке
- Оценивает вероятность того, что текст был создан ИИ
- Предоставляет конкретные рекомендации по улучшению на русском языке
- Может генерировать "улучшенную" версию документа с более естественным языком
- Сохраняет отчеты анализа в формате JSON

## Использование

### Командная строка

```bash
# Базовый анализ
python ai_detector.py document.docx

# Сохранить отчет анализа
python ai_detector.py document.docx -o report.json

# Генерация улучшенного документа
python ai_detector.py document.docx --fix --fix-output improved.docx
```

### Как модуль

```python
from ai_detector import AITextDetector

detector = AITextDetector()
results = detector.process_docx_file("example_russian.docx")

print(f"Вероятность ИИ: {results['analysis']['ai_probability']}%")
print(f"Найдено проблем: {len(results['analysis']['issues'])}")
```

## Как это работает

Детектор ищет общие паттерны в тексте, созданном ИИ:

- Чрезмерно формальный язык (например, "осуществлять" вместо "делать")
- Повторяющиеся переходные фразы (например, "Более того", "Кроме того")
- Шаблонные структуры предложений (например, "Во-первых", "Во-вторых", "В-третьих")
- Общие фразы ИИ (например, "В современном мире", "Следует отметить")

## Этические соображения

Этот инструмент предназначен для того, чтобы помочь пользователям улучшить свой письменный стиль и сделать контент, созданный с помощью ИИ, более естественным. Его следует использовать ответственно и этично, соблюдая академические и профессиональные стандарты честности.

## Поддержка русского языка

Программа специально нацелена на паттерны русского языка, которые часто встречаются в тексте, созданном ИИ. Она включает:

- Обнаружение формальных русских слов и фраз
- Распознавание повторяющихся структур предложений, типичных для русского текста ИИ
- Рекомендации по более естественным русским выражениям
- Возможности корректировки текста для замены шаблонных фраз более естественными альтернативами
