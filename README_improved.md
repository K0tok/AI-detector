# Улучшенный детектор ИИ-текста в DOCX файлах

Это улучшенная версия программы для обнаружения текста, сгенерированного ИИ, в DOCX файлах. Программа помогает пользователям идентифицировать потенциально ИИ-сгенерированный контент и предлагает улучшения, чтобы сделать текст более похожим на человечески написанный.

## Особенности улучшенной версии

1. **Улучшенный алгоритм обнаружения**:
   - Анализ статистики предложений (длина, разнообразие, сложность)
   - Анализ статистики слов (разнообразие словарного запаса, средняя длина)
   - Обнаружение пассивного залога и сложных структур
   - Поиск повторяющихся шаблонов и фраз

2. **Более точная оценка вероятности**:
   - Комбинированный подход: паттерны + статистический анализ
   - Более точное определение вероятности, что текст создан ИИ

3. **Расширенные предложения по улучшению**:
   - Рекомендации на основе статистического анализа
   - Более конкретные предложения по улучшению текста

4. **Дополнительные функции**:
   - Подробный режим анализа с детальной статистикой
   - Более качественные исправления текста

### Подробный анализ:
```bash
python ai_detector_improved.py document.docx --detailed
```

## Как определяется процент

Процент вероятности определяется на основе комплексного анализа:

1. **Паттерн-анализ** (50% веса):
   - Поиск характерных для ИИ фраз и структур
   - Проверка на избыточно формальный язык
   - Обнаружение повторяющихся переходов и шаблонов

2. **Статистический анализ** (50% веса):
   - Вариативность длины предложений
   - Разнообразие словарного запаса
   - Соотношение сложных и простых предложений
   - Частота использования формальных слов

Каждый фактор оценивается по шкале, суммируется и нормализуется в диапазоне 0-100%.

## Этическое использование

Этот инструмент предназначен для:
- Образовательных целей
- Помощи в написании более естественного текста
- Обнаружения потенциально ИИ-сгенерированного контента

НЕ используйте этот инструмент для:
- Обмана систем проверки на плагиат
- Нарушения академической честности
- Представления измененного ИИ-текста как оригинальной работы

## Использование в собственном коде

```python
from ai_detector_improved import AITextDetector

detector = AITextDetector()
results = detector.process_docx_file('document.docx')

print(f"Вероятность ИИ: {results['analysis']['ai_probability']}%")
print(f"Найдено проблем: {len(results['analysis']['issues'])}")
```
